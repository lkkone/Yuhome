# 咸鱼个人站 静态站点

海洋风格的个人首页导航页面，海洋背景、风格文字与卡片式站点展示、简约而不简单

## 预览
- 直接双击 `index.html` 本地预览

## 目录结构
```
info_v2/
  ├─ index.html
  ├─ styles.css
  ├─ app.js
  ├─ gilded_effect.min.js
  └─ images/
```

## 部署
- 支持任意静态托管（GitHub Pages / Vercel / Netlify / Nginx）
- GitHub Pages：Settings → Pages → 选择 main 分支 / root
